MALFORMED_RESPONSE

Description:

MALFORMED_RESPONSE

Indicates that the remote host has sent the local client back a well-formed response, but the response can't be understood.

This error occurs, for example, when the remote host sends a response in an XML grammar different from that expected by the local client.
