GeometryOptions

Description:

public enum GeometryOptions

Describes how the on-screen publishing dialog box should look and behave when an associated pane is installed in the on-screen publishing dialog box.

Content:

Enum values:

  • NONE - When the associated pane is installed, the on-screen publishing dialog box will be sized normally and will not allow the user to change its size.
  • EXTENDED_SIZE - If this bit is set, when the associated pane is installed, the on-screen publishing dialog box will grow to a larger size.
  • RESIZABLE - If this bit is set, when the associated pane is installed, the on-screen publishing dialog box will allow the user to change its size.
