install_login_wait_pane

Description:

public abstract void install_login_wait_pane ()

Works just like install_account_fetch_wait_pane but displays the static text “Logging in...“

As with install_account_fetch_wait_pane , this is a convenience method, but you should you use it provide to visual consistency and to let Shotwell handle internationalization. See the description of install_account_fetch_wait_pane for more information.

If an error has posted, the PluginHost will not honor this request.
