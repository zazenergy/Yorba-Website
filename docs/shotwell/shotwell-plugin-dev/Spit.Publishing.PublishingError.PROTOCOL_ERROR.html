PROTOCOL_ERROR

Description:

PROTOCOL_ERROR

Indicates that a communications channel to the remote host was opened and is active, but that messages sent to or from the remote host can't be understood.

This error occurs, for example, when attempting to interact with a RESTful host via XML-RPC.
