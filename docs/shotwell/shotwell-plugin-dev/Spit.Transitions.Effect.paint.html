paint

Description:

public abstract void paint (Visuals visuals, Motion motion, Context ctx, int width, int height, int frame_number)

Called when the effect needs to paint (i.e. an expose or draw event has occurred).

This call should not advance the state of the effect (i.e. it may be called more than once for the same frame).

Parameters:

ctx The Cairo context the Effect should use to paint the transition.
width The width (in pixels) of the Cairo surface.
height The height (in pixels) of the Cairo surface.
frame_number The one-based frame being drawn.
