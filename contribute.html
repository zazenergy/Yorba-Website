Together, we're building great software.
Here's how you can help:

Translation

Shotwell is translated into over 50 languages thanks to our dedicated team of translators. If you would like to help translate Shotwell into your language, you may do so through our project page on Transifex. To translate other projects, please contact info@yorba.org.

Testing

We're on a mission to squash bugs! Before filing a bug, please check on Yorba's Redmine instance to check if it's already been reported. If not, please create an account on Redmine and file a bug. Learn how to create the perfect bug report here.



Developing

Yorba is actively seeking volunteers to contribute code for Shotwell and Geary. All our projects are coded in Vala. Please read our coding conventions. Read more on Yorba Redmine about Geary or Shotwell.

Interested in working on any of our currently inactive projects? Awesome! Shoot us an email at info@yorba.org and we'll help you get started.

Plugins

Please read our coding conventions around plugin development if you'd like to write Plugins for Yorba.
